# 修改国内镜像

  • 配置源地址

    npm config set registry http://registry.npm.taobao.org/

  • 查看源地址受否正确的被更改

    npm config get registry

  • 取消镜像

    npm config delete registry

# 使用

  1. 初始化

    npm init -y

  2. 安装依赖

    npm install

  3. 如果出现系统禁止运行脚本，执行以下命令，没有出现则忽略

    set-ExecutionPolicy RemoteSigned

# 插件安装

全局安装 -g

项目依赖 --save

开发依赖 --sava-dev
